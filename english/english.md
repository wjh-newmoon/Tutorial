## 字母缩写释义

| 缩写     | 释义                                                     |
| -------- | -------------------------------------------------------- |
| **OCP**  | (The Open Closed Principle)开放-封闭原则                 |
| **JNDI** | (Java Naming and Directory Interface）Java命名和目录接口 |
|          |                                                          |

**OCP**:  (The Open Closed Principle)开放-封闭原则, 是说软件实体（类、模块、函数等等）应该可以扩展，但是不可修改。[ASD]
