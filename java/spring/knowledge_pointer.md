### IOC是什么

IOC (**Inversion of Control**) **控制反转**， **是一种设计思想**。在Java开发中，Ioc意味着将设计好的对象交给容器控制，而不是传统的在对象内部直接控制。

DI (**DI—Dependency Injection**) 依赖注入
